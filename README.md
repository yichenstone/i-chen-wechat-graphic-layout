# 一陈微信图文排版

#### 介绍
在微信公众号的运营中，图文排版是运营者主要工作之一。图文排版不仅是优化读者的视觉感受，也是运营者风格的体现。而风格则具体表现为运营者使用较为固定的文章格式、风格统一的图片等。使用Python搭建本地服务器与开发功能、Web开发用户界面、Socket进行两端通信的方法，设计了一款微信图文排版工具。运营者能够在对文字进行简单处理的基础上，实现高效图文排版。
